Cylindrical¶

A cylindrical joint creates a point that is translated along an axis defined in the reference frame, and a frame that is rotated relative to an existing frame about the same axis.

Keyword Arguments

param name

string

Name given to the joint, default is uppercase characters in alphabetical order (A, B, C, …)

param ref_point_key

string

Start point from which the vector component of the joint is defined

param ref_frame_key

string

Reference frame in which the translational and rotational displacement components are defined

param is_form

string

‘constant’ or ‘degree of freedom’

param motion_axis

string

‘Z’, ‘Y’, or ‘X’. Represents the axis in which rotation and translation are defined,

Initial Values

param initial_t

numeric

Initial value in the translational coordinate

param initial_r

numeric

Initial value for rotation

param rate_t

numeric

Initial speed in the translational coordinate

param rate_r

numeric

Initial speed for rotation

Spring Parameters

param kt

numeric

Stiffness in the translational coordinate

param kr

numeric

Stiffness for rotation

param ct

numeric

Damping in the translational coordinate

param cr

numeric

Rotational damping

param equilibrium_t

numeric

Equilibrium distance in the translational coordinate

param equilibrium_r

numeric

Equilibrium angle
