Rigid Body Box
==============

A rigid body box is a rectangular prism of uniform density. The box has length defined in the X-axis, width defined in the Y-axis, and height defined in the Z-axis, where (X, Y, Z) are the basis vectors of the body fixed frame.

**Keyword Arguments**

:param name: string

Key to be used when referencing this box.

:param ref_point_key: string

Key of the point to which this box is attached, either at its CG, aft, or forward ends.

:param ref_frame_key: string

Key of the frame to which this box orientation is defined.

:param density: numeric

Uniform density of this box.

:param length: numeric

Dimension of this box in the X-axis of the body-fixed frame.

:param width: numeric

Dimension of this box in the Y-axis of the body-fixed frame.

:param height: numeric

Dimension of this box in the Z-axis of the body-fixed frame.

:param attach_to: string

CG, fore, or aft

:return:
